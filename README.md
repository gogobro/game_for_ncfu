# Horror Rooms Prototype

Готовый Java-проект с:
- хоррор-комнатами
- дверями
- двумя монстрами
- атакой кулаком
- плавным движением по зажатию клавиш
- встроенными интернет-ассетами в ресурсах JAR

## Запуск на Windows
1. Установи Java 17+ или новее
2. Дважды нажми `run.bat`

## Запуск без консоли
Двойной клик по `run_silent.vbs`

## Запуск на macOS / Linux
```bash
chmod +x run.sh
./run.sh
```

## Управление
- WASD — движение
- E — открыть / закрыть дверь рядом
- SPACE или F — удар кулаком
- R — рестарт после смерти или победы
